# AluraPhyton
Durante este desafío, ayudarás al Sr. Juan a decidir qué tienda de su cadena Alura Store debe vender para iniciar un nuevo emprendimiento. Para ello, analizarás datos de ventas, rendimiento y reseñas de las 4 tiendas de Alura Store. El objetivo es identificar la tienda menos eficiente y presentar una recomendación final basada en los datos.
---
Proyecto de análisis de datos utilizando Python, pandas y matplotlib.
Se analizan ingresos, categorías más vendidas, productos más y menos vendidos.
También se calcula la calificación promedio y el costo de envío promedio por tienda.
Requiere Python 3.8 o superior.
Instalar dependencias con: pip install pandas matplotlib
Colocar los archivos tienda_1.csv, tienda_2.csv, tienda_3.csv y tienda_4.csv en la misma carpeta.
Ejecutar el archivo .ipynb en Google Colab o Jupyter Notebook.
Si ocurre FileNotFoundError, verificar la ubicación de los archivos.
Si ocurre KeyError, revisar los nombres de las columnas con print(df.columns)
